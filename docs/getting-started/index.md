# 快速开始总览

本章节帮助你在 10–20 分钟内完成本地启动，并理解最小可运行链路。

## 你将获得什么

- 能运行 `main.py` 并确认服务健康状态。
- 能启动 Web 前端并完成基础配置访问。
- 能执行关键测试，确认改动不会破坏核心能力。

## 阅读顺序

1. [安装与环境准备](./installation.md)
2. [启动模式与最小验证](./running.md)

## 关键入口文件

- 启动入口：`main.py`
- Runtime 内核：`src/runtime/brain.py`
- 配置系统：`src/runtime/config_manager.py`
- 管理 API：`src/tools/admin_api.py`
