# 机械结构设计 (Mechanical Design)

> 状态: 待设计

## 5-DOF 情感表达机械臂
- J1: 基座旋转 (±180°, 配滑环)
- J2: 肩部俯仰 (0°~120°)
- J3: 肘部俯仰 (0°~135°)
- J4: 头部偏航 (±70°)
- J5: 头部俯仰 (±60°)

## 尺寸
- 底座: φ60mm
- 大臂/小臂: 各 110mm
- 头部: φ60mm 球体
- 总高: ~25-30cm

## 执行器
Feetech STS3215 串行总线舵机 × 5

## 文件规划
- `stl/` — 3D 打印件 (FDM/SLA)
- `step/` — 装配体工程文件
- `drawings/` — 2D 工程图

详细设计见 `doc/mechanical_design.md`
